| # | PRD 位置 | 描述 | 遗漏点 | 建议新增用例 |

| A-1 | §1.4 / 附录 C | 权限操作粒度 6 种：查看 / 编辑 / 管理 / 操作 / 导出 / **结算** | 目前只有零散校验，未为每种粒度单独设立用例 | 新增 `TC-A-14-020~030`：对同一菜单（如收益管理）分别配置 view-only / operate-only / settle 三种角色，验证按钮/接口级差异 |
