# 管理端测试用例

对应 PRD §7.3。覆盖 18 个页面：运营看板、用户组、作业、用户档案、合同台账、评审绩效、账号与权限、平台管理员、角色管理、歌曲库、批量下载、ISRC、发行渠道、发行状态确认、收益管理（5 模块）、内容管理、系统设置（5 子模块）、操作日志。

## 通用前置

- 登录超管 `admin / Abc12345`（或具备对应权限 Key 的管理员）
- 测试数据：2 个用户组、10 位创作者（含各实名状态）、2 位评审、10 首不同状态作品、3 个导入批次、若干 `suspect/pending` 映射

## 通用约定

- 管理端所有写操作均须写入 `operation_logs`（P2-2 已全覆盖，见对应用例 TC-A-18-*）
- 权限校验：菜单项级别对应 `admin.{menu}.view/operate/manage/export/settle` 等 Key
- 列表默认 `page_size=20`
