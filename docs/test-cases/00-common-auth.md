# 公共测试用例 · 认证

覆盖：创作者注册、三端登录、短信验证码、重置密码、登出。
对应 PRD：1.3（用户组与注册机制）、1.5（三端架构）、10.6（限流 / SLA）。

## 前置数据

- 有效邀请码 `TESTCODE8`（所属组 `active`）
- 暂停组邀请码 `PAUSED01`（所属组 `paused`）
- 已注册创作者手机号 `13800138000`
- 超管账号 `admin / Abc12345`
- 评审账号 `reviewer01 / Abc12345`
