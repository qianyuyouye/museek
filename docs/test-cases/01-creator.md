# 创作者端测试用例

对应 PRD §7.1（含 10 个页面）：首页、自由上传、作业提交、我的作品库、我的收益、课程中心、作品广场、我的学习、消息中心、个人中心。

## 通用前置

- 登录创作者账号 `13800138000 / Abc12345`
- 具备权限：`student.*` 全部查看/操作
- 至少存在：1 首作品（各状态各 1 条）、1 个进行中作业、3 条消息、3 门课程
