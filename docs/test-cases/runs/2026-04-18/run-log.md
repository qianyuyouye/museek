## 已跑通的端到端主链路

✅ **评审→发行闭环**：
1. 评审 `reviewer01` 登录 → 待评审队列 → 评审午夜电波（91 分/强推）
2. 歌曲状态 pending_review → **ready_to_publish**
3. Admin 登录 → 歌曲库管理 → 待发行 Tab
4. 未绑 ISRC 点确认发行 → 后端阻断（状态不变）
5. SQL 绑 ISRC → 再点确认发行 → **published**
6. operation_logs 新增 `song_publish` 记录

✅ **用户组管理**：创建 `E2ETEST1` → operation_logs `create_group` 记录

✅ **作品状态机**：
- `pending_review → reviewed`（强推+总分<80）
- `pending_review → ready_to_publish`（强推+≥80）
- `ready_to_publish → published`（三条件满足）
- `needs_revision` 创作者端显示"修改并重新提交"按钮
